# Контекстная алгебра памяти (CMA)

## Новая математическая система 2026

---

## Что это?

**Контекстная алгебра памяти (CMA)** — это принципиально новая математическая система, где результат бинарной операции зависит от истории предшествующих вычислений.

### Ключевое отличие от всей существующей математики

В классической алгебре: `a + b` всегда даёт одинаковый результат, независимо от того, какие операции выполнялись ранее.

В CMA: `a ⊕_C b` может давать разные результаты в зависимости от контекста `C` (истории операций).

---

## Почему это принципиально новое?

1. **Нет аналогов**: Ни одна существующая математическая система не использует зависимость результата операции от истории вычислений
2. **Нет близких концепций**: Даже неассоциативные алгебры (кватернионы, октавы) не зависят от истории
3. **Уникальная аксиоматика**: 5 аксиом, полностью определяющих новую алгебраическую систему

## Быстрый старт

```python
from cma_core import create_cma_from_numbers

# Создаём CMA для сложения с памятью глубиной 3
cma = create_cma_from_numbers('add', k=3)

# Первая операция - без влияния истории
result1 = cma.operate(5, 3)  # = 8

# Вторая операция - с влиянием первой
result2 = cma.operate(2, 2)  # = 4 + влияние контекста

# Третья операция - с влиянием первых двух
result3 = cma.operate(1, 1)  # = 2 + влияние контекста
```

## Математическая структура

### Аксиомы

1. **Существование нейтрального элемента**: `a ⊕ e = a`, `e ⊕ a = a`
2. **Зависимость от контекста**: Результат зависит от истории
3. **Затухание памяти**: При `k=0` — классическая алгебра
4. **Монотонность влияния**: Более поздние операции влияют сильнее
5. **Завершённость контекста**: После операции контекст обновляется

### Теоремы

1. **Теорема о затухании**: Влияние ранних операций уменьшается с глубиной
2. **Теорема о единственности нейтрального элемента**: Ровно один `e`
3. **Теорема о контекстной независимости**: При `k=0` — изоморфизм классической алгебре
4. **Теорема о вычислительной сложности**: `O(k)` дополнительных ресурсов
5. **Теорема о неподвижной точке**: Существует `x: x ⊕ a = x`

---

## Типы CMA

- **LinearCMA**: Линейное влияние контекста
- **MultiplicativeCMA**: Мультипликативное влияние
- **NeuralCMA**: Нейронное (нелинейное) влияние
- **EvolutionaryCMA**: Для эволюционных алгоритмов
