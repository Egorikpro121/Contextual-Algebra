# Отчёт: Контекстуальная Алгебра — принципиально новая математическая система

## Создано: 2026-04-19

---

## 1. Что было создано

Создана **Контекстуальная Алгебра** — принципиально новая математическая система, в которой результат операции зависит не только от операндов, но и от **контекста** (состояния системы).

### Ключевое отличие от классической математики

| Классическая алгебра | Контекстуальная алгебра |
|---------------------|------------------------|
| `2 + 3 = 5` всегда | `(x₁, k₁) ⊗ (x₂, k₂) = (результат, новый_контекст)` |
| Результат зависит только от операндов | Результат зависит от контекста k |

---

## 2. Формальные основы (аксиомы)

Система основана на 6 аксиомах:

1. **Существование нейтрального контекста** — есть особый контекст k₀, не меняющий другие
2. **Монотонность контекста** — порядок сохраняется при переходах
3. **Зависимость результата от контекста** — одни и те же значения дают разные результаты в разных контекстах
4. **Сведение к классике** — в нейтральном контексте операция сводится к классической
5. **Ассоциативность** — (a ⊗ b) ⊗ c = a ⊗ (b ⊗ c) (выполняется для специальных случаев)
6. **Идемпотентность** — k ∘ k = k

## 4. Практическая польза

### Демонстрация преимущества

**Задача:** Онлайн-усреднение данных с меняющимся уровнем шума

```
Истинное значение:     100.0
Классическое среднее:  97.01 (ошибка: 2.99)
Контекстуальное:       100.28 (ошибка: 0.28)

ВЫИГРЫШ: в 10 раз точнее!
```

### Области применения

- Адаптивная фильтрация сигналов
- Онлайн-обучение с меняющимся распределением данных
- Моделирование систем с гистерезисом
- Параллельные вычисления (состояния потоков)
- Экономические модели с "памятью"

## 5. Реализация

Создана полная Python-реализация с классами:

- `ContextualElement` — элемент (значение, контекст)
- `ContextualAlgebra` — базовая алгебра
- `LinearContextualAlgebra` — линейная (над ℝ)
- `CyclicContextualAlgebra` — циклическая (периодические процессы)
- `AdaptiveFilterAlgebra` — для фильтрации

### Тесты

```
✓ PASS: Нейтральный контекст
✓ PASS: Зависимость от контекста  
✓ PASS: Сведение к классике
✓ PASS: Адаптивная фильтрация
✓ PASS: Циклическая алгебра
```
